Raspberry Pi 3B Proxy Server 配置

2017-05-10
Lantern, Raspberry, Ubuntu

自从发现树莓派可以运行 ubuntu 后就一直再想能不能在上面跑 lantern ,从而实现低功率全局梯子，在看了大量入门帖子后决定入手一试，反正也不贵，我入手的是 e14 国行版，和英国产的应该都没区别的，配件另外买了带开关的电源、16G TF 卡、一个原装外壳、两铜一铝的散热片，其实散热片买一铜一铝就够了，第三个散热完全用不上的。

1.准备工作

树莓派支持很多中操作系统，用得最多的就是原厂出的 Raspbian ,另外一个比较出名的就是 ubuntu ,这两个我都比较过，虽然 Raspbian 运行速度很快，但是很多工具都是没有界面的，对新手使用很不方便，而 ubuntu 则更容易上手，基本用到的工具都是有图形界面的，就跟用 windows 差不多，就是运行起来速度跟 Raspbian 完全没法比啊，不过我只是用来跑 lantern 的基本也就无所谓了。

1.1 下载系统

选好系统就可以开始了，先下载好 ubuntu mate 的镜像文件，可以在 这里 下载，选 16.04.2 LTS 下的 Raspberry Pi 分支，下载回来要解压出来里面的 img 文件。

1.2 镜像写入工具

Win32 DiskImager 点这里下载

2. 安装系统

直接运行 win32 DiskImager ，在 Image File 里选择解压出来的 img 文件，Device 那里选择 U 盘盘符然后点击 Write 就可以了，下面有进度条，完成会有个 Successful 弹窗。

3. 开机

插上 TF 卡接好电源和 HDMI 输出，还有鼠标键盘，网络的话可以用 wifi 或者网线都可以，都接好就可以开机了，通电后主板上的信号灯会亮。

进入系统后会要进行一些简单配置，设置好账户名和密码，时区选择 shanghai 就好了，基本和一般的 ubuntu 配置一样，最后等待系统安装完成就会进入桌面了。

Fedora 25 桌面配置

2017-04-16
Centos, Fefora, Linux

配置

添加国内源

在下载和更新软件时，系统自带的软件源还是比较慢的，添加国内的第三方软件源可提高下载速度。依次在终端输入如下命令：

自动选择最快源插件

sudo dnf install yum-fastestmirror

安装下载加速插件

dnfinstall yum-presto

更新系统

sudo dnf update

安装鼠标右键在终端中打开

yum install nautilus-open-terminal

安装 Nvidia 显卡驱动

https://rpmfusion.org/Howto/nVidia

美化

安装 gnome-tweak-tool

sudo dnf install gnome-tweak-tool

把系统自带的文件图标改小

gsettings set org.gnome.nautilus.icon-view default-zoom-level small

安装Theme

主题： OSX-Arc-Collection

图标： Arc-OSX-Icons

Dock： Dash to Dock
